# lein-webutils

A Leiningen plugin to help my personal web dev projects.

## Usage

Put `[lein-webutils "0.1.0-SNAPSHOT"]` into the `:plugins` vector of your project.clj.

    $ lein web compile
    $ lein web watch
    $ lein ring server
